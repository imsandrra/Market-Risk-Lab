# Market Risk Lab – PwC
Repozytorium zawiera rozwiązania zadań domowych z kursu **Market Risk Lab** prowadzonego przez PwC.

## Struktura
| Folder | Temat |
|--------|-------|
| `ZD1/` | Dane rynkowe – pobieranie, czyszczenie, stopy zwrotu, korelacje |
| `ZD2/` | Value at Risk – metoda historyczna i parametryczna |
| `ZD3/` | Wycena opcji europejskich – Black-Scholes vs Monte Carlo |

## Dane
Akcje sektora finansowego Stanów Zjednoczonych (GS, C, AIG, BAC, MS), okres 2006-08-01 – 2008-07-31.  
ZD3 skupia się na Goldman Sachs (GS) jako instrumencie bazowym opcji.
